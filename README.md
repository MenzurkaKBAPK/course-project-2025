# Сервис бронирования коворкинга

## Описание

Веб-приложение для бронирования рабочих мест в коворкинге. Пользователи могут регистрироваться, входить, бронировать, просматривать и управлять своими бронями. Администраторы управляют пользователями и местами.

## Стек

- **Frontend:** React (SPA)
- **Backend:** Spring Boot (Java)
- **Auth:** Go (JWT)
- **База данных:** PostgreSQL
- **Сборка и запуск:** Docker + Docker Compose

## Возможности

### Пользователи:
- Регистрация и вход
- Бронирование и отмена
- История бронирований
- Профиль

### Администраторы:
- Управление пользователями
- Добавление/удаление мест
- Просмотр всех броней

## Дополнительно

- Все сервисы взаимодействуют через REST API
- Авторизация через JWT
- Swagger-документация
- Контейнеризация через Docker
